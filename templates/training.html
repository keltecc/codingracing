Q. What I need to do?

A. You need to type code on the right side as fast as you can!

Q. How you calculate typing speed?

A. We add penalty to spent time. After this, the length of the program is divided by the total time and we get speed!

Q. How is the penalty time calculated?

A. Penalty time is calculated as fivefold Levenshtein distance between your and source code.

Q. I need write code exactly character-in-character?

A. No, all whitespace characters (including tabs and line breaks) are ignored. So you can write everything in one line.

Q. Can I optimize program's code? For example replace x = x + 1 to x += 1.

A. No, you can't. You need to write exactly the same program.
